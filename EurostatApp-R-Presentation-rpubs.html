Data:-

The data used for developing Eurostat-Road Accident picked from the Eurostat Package. Eurostat package provide open data to R tools, Data search, download, manipulation and visualization. In this package updated data on various parameter is given. I have used road accident victims data to build my shiny app. You can see original dataset on this web site :- People killed in road accidents (tsdtr420)

App description and Screen shot of App:-

I tried to build an App which shows people killed in the road accident by year and countries in the Europe. Using this App we can easly analyze that which country is keen about road accident & safety of people. For the sake of handyApp, I have add some features such as:-

  • Drop-Down Button (to choose country or Year).
  • Radio Button (Shows: Summary & Data table).
  • Slider (IT slide manually and automatically).
